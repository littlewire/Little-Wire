Introduction

This is the doxygen documentation of the C++ library of Little-Wire project. Other language interfaces (C,C#,Processing) of the Little-Wire uses mostly the same names and usages. Most function names and usage are chosen as similar as possible to the Arduino library functions for to lower the learning curve.

Installation

Step 1

Download Libusb

Step 2

Copy the files under library to your working folder and include littleWire.h to your program.

Step 3

Look to the provided examples for further help.
