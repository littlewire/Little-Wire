Little Wire

Little Wire is multi-featured USB controlled open source hardware tool packed in a minimal form factor designed by ihsan Kehribar. This documentation explains how you can utilize features of the Little Wire via computer inteface library written in C language.

Current version is: v1.3

For additional information about usage of this library, please look to the examples folder.

Please submit your bug reports / pull requests from GitHub: http://github.com/littlewire/Little-Wire
